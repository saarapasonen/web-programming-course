# Chat room app
Keskustelusovellus; 

- Sovelluksessani käyttäjät voivat kirjautua sisään ja ulos sen jälkeen, kun on tehnyt uuden tilin ja rekisteröitynyt. 
- Etusivulla käyttäjät voivat selata eri keskustelualueita ja nähdä niiden ketjujen ja viestien määrän ja viimeisimmän viestin ajankohdan.
- Käyttäjillä on mahdollisuus luoda uusia keskusteluketjuja antamalla otsikko ja aloitusviesti.
- Lisäksi käyttäjät voivat osallistua olemassa oleviin keskusteluihin kirjoittamalla uusia viestejä.
- Sovelluksessa on myös hakutoiminto, jonka avulla käyttäjät voivat etsiä viestejä annetun haun (sanan) perusteella.
- Käyttäjät voivat muokata omia luomiaan ketjuja sekä poistaa omia viestejään tarvittaessa. 
- Ylläpitäjillä taas on valtuudet hallita keskustelualueita, mukaan lukien niiden lisääminen ja poistaminen tarpeen mukaan.
- He voivat myös luoda salaisia keskustelualueita ja päättää, ketkä käyttäjät saavat niihin pääsyn. 
- Sovellus tarjoaa käyttäjilleen helppokäyttöisen alustan keskusteluun ja ylläpitäjille tehokkaat työkalut keskustelualueiden hallintaan ja valvontaan.
